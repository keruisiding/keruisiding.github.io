learn about mac

终于进入了主题了，拿到mac没有多久，但是深深感受到mac的好用了。mac 下载一个xcode是十分必要的，即使用不到xcode。xcode自带的许多功能在
自后的很多项目中可以用到，最经典的就是git和command line。
1、之前在使用mac 和GitHub连接的时候的确花了不少时间。首先是，建立的rsa.pub文件，在终端下面可以进入到文件夹.ssh，但是在finder里面缺
找不到此文件夹，于是打开它就出现了问题，尝试了使用neno指令打开，但是还不熟练，只能看到一行。最后感谢度娘上的经验，先后了解了mac显示隐藏的文件夹，最后右键finder 前进到文件夹，输入文件路径，进入到了 .ssh文件夹下面copy到了rsa.pub文件的内容。
2、将自己的mac和GitHub账号关联，由于自己有两个GitHub账号，目前在连接的时候还没有找到快捷的办法。每次初始化了git然后确认自己的用户名
然后在连接上传。
